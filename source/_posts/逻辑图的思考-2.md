### 1.符合`第一镜像线`特性
```
{(0,0),
 (0,1)} 与,
{(1,0),
 (0,0)} 或非, 
 
{(1,1),
 (1,0)} 与非,
{(0,1),
 (1,1)} 或,  
 
三种转换方式
组合，不相干逻辑图的后置取反，第二镜像线对面逻辑图的前置取反
```

### 2.符合`第二镜像线`特性
```
{(1,0),
 (1,1)} A蕴含B, (即A为真时,B值决定结果真假.A为假时,结果全为真)，
{(0,1),
 (0,0)} (A蕴含B)非, 
 
{(1,1),
 (0,1)} B蕴含A,
{(0,0),
 (1,0)} (B蕴含A)非, 
 
两种转换方式 
第一镜像线对面逻辑图的AB点互换，不相干逻辑图的后置取反
```

### 3.同时符合
```
{(0,1),
 (1,0)} 异或, 
{(1,0),
 (0,1)} (异或)非, 
```

### 4.双线中,一线常开
```
{(0,1),
 (0,1)} A线性,
{(1,0),
 (1,0)} A非, 
 
{(0,0),
 (1,1)} B线性,
{(1,1),
 (0,0)} B非, 
 
两种转换方式
第一镜像线对面逻辑图的AB点互换，不相干逻辑图的后置取反
```

### 5.双线中,两线相同
```
{(0,0),
 (0,0)} 常开, 
{(1,1),
 (1,1)} 常闭

可以由前三种类型组合。
```
